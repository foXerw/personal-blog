# 📝 发布文章指南

## 快速开始

### 步骤 1：创建 Markdown 文件

在 `posts/` 目录下创建新文件，命名格式：`任意英文名.md`

**推荐命名**：使用文章标题的拼音或英文
```
posts/
├── hello-world.md
├── minimalism.md
├── frontend-art.md
└── 你的新文章.md
```

---

### 步骤 2：编写文章内容

复制下面的模板到你的新文件：

```markdown
---
title: 你的文章标题
date: 2026-03-08
updated: 2026-03-08
categories: [技术]
tags: [标签 1, 标签 2]
cover: 
draft: false
featured: false
description: 文章描述，用于 SEO 优化
---

## 第一个标题

这里是文章内容...

### 代码示例

```javascript
const hello = 'world';
console.log(hello);
```

### 列表

- 项目 1
- 项目 2
- 项目 3

### 引用

> 这是一段引用。

### 图片

![描述](images/posts/your-image.jpg)
```

---

### 步骤 3：更新文章索引

编辑 `data/posts.json`，在 `posts` 数组中添加新文章信息：

```json
{
  "posts": [
    {
      "slug": "你的文件名（不含.md）",
      "title": "你的文章标题",
      "date": "2026-03-08",
      "updated": "2026-03-08",
      "categories": ["技术"],
      "tags": ["标签 1", "标签 2"],
      "excerpt": "文章摘要，显示在首页...",
      "cover": "",
      "featured": false,
      "draft": false,
      "readingTime": 5
    }
  ],
  "categories": [...],
  "tags": [...],
  "generatedAt": "2026-03-08T00:00:00.000Z"
}
```

**重要字段说明**：

| 字段 | 说明 | 示例 |
|------|------|------|
| `slug` | 文章标识，**必须与文件名一致** | `"hello-world"` |
| `title` | 文章标题 | `"欢迎来到我的博客"` |
| `date` | 发布日期（YYYY-MM-DD） | `"2026-03-08"` |
| `categories` | 分类数组 | `["技术", "生活"]` |
| `tags` | 标签数组 | `["JavaScript", "前端"]` |
| `excerpt` | 摘要（首页显示） | `"这是文章简介..."` |
| `featured` | 是否置顶（true/false） | `true` |
| `readingTime` | 阅读时间（分钟） | `5` |

---

### 步骤 4：添加配图（可选）

如果有封面图或文章配图：

1. 将图片放入 `images/posts/` 目录
2. 在 Markdown 中使用：`![描述](images/posts/your-image.jpg)`
3. 在 `posts.json` 中设置 `cover` 字段

---

### 步骤 5：测试预览

1. 刷新浏览器：`Cmd+Shift+R`
2. 在首页查看新文章卡片
3. 点击文章链接查看完整内容
4. 检查深色/浅色主题下的显示效果

---

### 步骤 6：提交到 Git

```bash
cd ~/Projects/personal-blog

# 查看所有改动
git status

# 添加新文件
git add posts/你的文章.md
git add data/posts.json

# 提交
git commit -m "feat: 发布新文章 - 你的文章标题"

# 推送（如果已配置远程仓库）
git push
```

## 📊 分类和标签管理

```json
"categories": [
  {"name": "技术", "count": 2},
  {"name": "新分类", "count": 1}  // 新增
]
```

在 `data/posts.json` 的 `tags` 数组中添加：

```json
"tags": [
  {"name": "JavaScript", "count": 1},
  {"name": "新标签", "count": 1}  // 新增
]
```

## ⚠️ 常见问题

### Q: 文章链接 404？
**A**: 检查 `slug` 是否与文件名一致（不含 `.md`）

### Q: 文章不显示？
**A**: 检查 `draft` 是否为 `false`

### Q: 首页看不到摘要？
**A**: 确保 `data/posts.json` 中 `excerpt` 字段有内容

### Q: 如何置顶文章？
**A**: 设置 `featured: true`，并在 CSS 中调整样式

## 🎯 最佳实践

1. **文件名**：使用英文或拼音，避免中文
2. **摘要**：保持 50-100 字，吸引读者
3. **标签**：每篇文章 2-5 个标签
4. **分类**：每篇文章 1-2 个分类
5. **配图**：压缩图片，优化加载速度
6. **SEO**：填写 `description` 字段
